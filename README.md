## ✨ Why Klip

Most video downloaders are either browser extensions that break every month, or clunky Python scripts you have to babysit. Klip is a real desktop app — fast startup, no terminal windows, no dependencies to install.

- 🎬 **Any quality** — 4K down to audio-only, your call
- ✂️ **Trim before you download** — grab just the clip you need, skip the rest
- 📋 **Playlists** — full playlists or custom ranges
- 🕘 **History** — see everything you've downloaded, when you want it
- 📎 **Clipboard auto-detect** — copy a link, Klip notices
- ⚙️ **Configurable concurrency** — control how many downloads run at once

## 🌐 Supported Platforms

YouTube · TikTok · Instagram · Twitter/X · Facebook · Reddit · Twitch · Vimeo · **+ thousands more**

## 🚀 Installation

1. Grab the latest installer from [**Releases**](https://github.com/gitttsarya/Klip/releases/latest)
2. Run it
3. Paste a URL, hit download

`ffmpeg` and `yt-dlp` come bundled — nothing else to install.

## 🛠️ Tech Stack

| Layer | Tech |
|---|---|
| Shell | [Tauri](https://tauri.app/) (Rust) |
| UI | React + TypeScript |
| Engine | yt-dlp + ffmpeg (bundled sidecars) |
